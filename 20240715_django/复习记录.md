# 二、如何安装mysql数据库

## 1. 下载 MySQL 安装包

- 访问 MySQL 官方下载页面：https://dev.mysql.com/downloads/mysql/
- 选择 "Windows (x86, 64-bit), MSI Installer" 版本

## 2. 安装 MySQL

### 运行安装程序
- 选择安装类型：
  - **Developer Default**：适合开发人员的默认安装
  - **Server Only**：只安装 MySQL 服务器
  - **Custom：自定义安装（推荐选择此项以精确控制安装组件）**

### 配置过程
- 在安装过程中会进行以下配置：
  1. **MySQL Server 配置**
     - 选择配置类型（Standalone MySQL Server）
     - 设置端口号（默认 3306）
     - 配置 MySQL root 用户密码（请务必记住此密码）
  2. **Windows Service 配置**
     - 设置 MySQL 作为 Windows 服务自动启动
  3. **Apply Configuration**
     - 应用配置并完成安装


## 3. 配置环境变量（可选但推荐）

为了能在任何位置使用 mysql 命令，需要将 MySQL 的 bin 目录添加到系统 PATH 环境变量中：

1. 找到 MySQL 安装目录
2. 右键"此电脑" → "属性" → "高级系统设置"
3. 点击"环境变量"
4. 在"系统变量"中找到 "Path" 并编辑
5. 添加 MySQL 的 bin 目录路径

# 三、启动/停止/重启：mysql或redis

## 1. 通过服务管理器重启

- 按 `Win + R` 打开运行对话框
- 输入 `services.msc` 并回车
- 在服务列表中找到 `MySQL84` 或者`redis`
- 如果服务正在运行，右键点击选择「重新启动」
- 或者先右键选择「停止」，等待停止完成后，再右键选择「启动」

## 2. 通过命令行重启

以管理员身份打开cmd

```bash
# 停止服务
net stop mysql84
net stop redis
# 启动服务
net start mysql84
net start redis
#redis端口：6379
#mysql端口：3306
```
