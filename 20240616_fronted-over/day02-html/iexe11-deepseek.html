DIV居中方法大全

在HTML/CSS中，有多种方法可以使DIV元素居中。下面展示了6种常用的居中技术，点击左侧菜单查看每种方法的实现。

Flexbox 居中法

使用Flexbox轻松居中
.parent {
    display: flex;
    justify-content: center;
    align-items: center;
    height: 100%;
}

.child {
    /* 子元素不需要特殊样式 */
}

Grid 居中法

使用Grid布局实现居中
.parent {
    display: grid;
    place-items: center;
    height: 100%;
}

.child {
    /* 子元素不需要特殊样式 */
}

绝对定位 + Transform 居中法

使用绝对定位和Transform实现居中
.parent {
    position: relative;
    height: 100%;
}

.child {
    position: absolute;
    top: 50%;
    left: 50%;
    transform: translate(-50%, -50%);
}

Auto Margin 居中法

使用Auto Margin实现水平居中
.parent {
    height: 100%;
}

.child {
    width: 200px; /* 需要指定宽度 */
    margin: 50px auto; /* 上下边距50px，左右自动 */
}

Text-Align 居中法

使用Text-Align实现行内块元素居中
.parent {
    text-align: center;
    height: 100%;
    padding-top: 50px; /* 用于垂直方向上的粗略调整 */
}

.child {
    display: inline-block;
    text-align: left; /* 恢复内部文本对齐 */
}

Table-Cell 居中法（传统方法）

使用Table-Cell实现居中
.parent {
    display: table;
    width: 100%;
    height: 100%;
}

.middle {
    display: table-cell;
    vertical-align: middle;
    text-align: center;
}

.child {
    display: inline-block;
}
