CSS媒体查询详解

@media (max-width: 480px) 的含义与应用

媒体查询是什么？

媒体查询是CSS3的一个特性，它允许内容适应不同的屏幕尺寸和设备类型。

@media (max-width: 480px) 表示当屏幕宽度小于或等于480像素时，应用其中的CSS规则。

  • 中等屏幕(≤768px)：每行显示3个导航项
  • 小屏幕(≤480px)：每行显示2个导航项
  • 超小屏幕(≤320px)：每行显示1个导航项

/* 平板设备样式 */
@media (max-width: 768px) {
    .nav-item {
        flex: 0 0 33.33%;
    }
    
    .brand-grid {
        grid-template-columns: repeat(2, 1fr);
    }
}

/* 手机设备样式 */
@media (max-width: 480px) {
    .nav-item {
        flex: 0 0 50%;
    }
    
    .brand-grid {
        grid-template-columns: 1fr;
    }
}

/* 非常小的手机设备样式 */
@media (max-width: 320px) {
    .nav-item {
        flex: 0 0 100%;
    }
}
